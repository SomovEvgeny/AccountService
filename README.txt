Клиент серверное приложение работает на базе RMI 
для запуска сервера, сначала перейдите в папку 

C:\Users\21\IdeaProjects\AccountService\out\production\AccountService

и запустите RMI регистр 

start rmiregistry -J-Djava.rmi.server.useCodebaseOnly=false

затем можно запускать сервер

При запуске клиента через командную строку следует указать следующие па-
раметры 

java -cp Client rCount wCount idGetList idAddList

rCount - кол-во потоков в которых запускается комманда getAmmount

wCount - кол-во потоков в которых запускается комманда addAmount

idGetList  - диапазон id  которые используются для getAmmount комманд (формат: 2 числа разделенных "-")

idAddList - диапазон id  которые используются для addAmmount комманд (формат: 2 числа разделенных "-")

Пример :

java -cp Client 5 5 1-5 1-5
